# Getting Started with Stripe Guardian

Welcome to Stripe Guardian! This guide will help you get started with protecting your Stripe Connect platform from fraud.

## What is Stripe Guardian?

Stripe Guardian is a fraud detection and prevention system for Stripe Connect platforms. It monitors payout activity and other account behaviors to detect suspicious activities that might indicate fraud.

<Alert type="info">
  Stripe Guardian is designed to complement Stripe Radar, focusing specifically on payout fraud
  patterns that Radar doesn't typically cover.
</Alert>

## Quick Setup

Follow these steps to set up Stripe Guardian:

1. **Sign up for an account**
2. **Connect your Stripe account**
3. **Configure your alert preferences**
4. **Set up notification channels**

## Key Features

### Real-time Fraud Detection

Stripe Guardian monitors your connected accounts in real-time, detecting suspicious activities as they happen. This allows you to respond quickly to potential fraud.

### Customizable Rules

You can customize the rules that trigger alerts to match your business needs. This helps reduce false positives and ensures you're only alerted about meaningful events.

<Alert type="warning" title="Important">
  While you can customize rules, we recommend starting with the default configuration, which is
  designed to catch the most common fraud patterns.
</Alert>

### Multi-channel Notifications

Get notified about potential fraud through multiple channels:

- Email
- Slack
- Microsoft Teams
- Webhook
